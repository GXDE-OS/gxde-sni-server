# GXDE SNI Server
This is GXDE SNI Server, its source code is adapted from Plasma workspace v6.7.1, commit id `58c75e75c40390f3793ffccc5e77bfa1cabeab88`.

Please note that this shim only supports Wayland because `deepin-daemon` handles SNI server correctly under X11.
